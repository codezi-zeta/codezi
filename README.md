## ⬇️ Setup

Requires [Node.js](https://nodejs.org/) to run (v14.15.0+).

Install the dependencies and open the terminal and then run the following command one by one.

```sh
$ git clone https://github.com/codezi-zeta/codezi.git
$ cd codezi
$ yarm
$ yarn start
```
